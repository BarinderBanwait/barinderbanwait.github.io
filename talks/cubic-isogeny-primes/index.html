$E/\mathbb{Q}$ elliptic curve.

Theorem (Mazur, 1978)

If $E/\mathbb{Q}$ is an elliptic curve admitting a rational $p$-isogeny, then $$p \in \{2,\,3,\,5,\,7,\,11,\,13,\,17,\,19,\,37,\,43,\,67,\,163\}.$$

Question (Cremona)

Mazur did this for $\mathbb{Q}$ in '78 — can you do it for any other number field?

↓ Mazur's list

Mazur's list

$\mathrm{IsogPrimeDeg}(\mathbb{Q}) = \{2, 3, 5, 7, 11, 13, 17, 19, 37, 43, 67, 163\}.$

Twelve primes. The largest, $163$, is famously connected to $\mathbb{Q}(\sqrt{-163})$ having class number one.

Why $d = 3$

  • A cubic field contains no imaginary quadratic subfield.
  • So the usual non-CM caveat is automatic.
  • Cleaner than $d = 2$.

The tool: isogeny-primes

Joint with M. Derickx.

Input: a number field $K$.
Output: a provably correct superset of $\mathrm{IsogPrimeDeg}(K)$.

Sweeping it across the LMFDB cubic fields gives a finite upper bound for strong uniformity in degree 3.

↓ what's inside

What's inside

  • Galois representations attached to $E[p]$.
  • Isogeny characters → constraints on $p$ from class field theory.
  • Auxiliary primes; Frobenius traces.
  • Output: a finite, often small, set of candidate $p$.

Sample run

$ sage isogeny_primes.py 'x^3 - x - 1'

Returns a finite superset of $\mathrm{IsogPrimeDeg}(K)$ for $K = \mathbb{Q}[x]/(x^3 - x - 1)$.

Strong uniformity mode: sage uniform_isogeny_primes.py 3.

The hard part: realization

For each candidate prime $p$, decide whether some cubic $K$ and $E/K$ witness $p$ as an isogeny degree.

Equivalently: for which $p$ does $X_0(p)$ admit a non-cuspidal cubic point?

↓ status on small primes

Status on small primes

  • $p = 73$: cubic points on $X_0(73)$ are finite and classified — benign.
  • $p \in \{23, 29, 31\}$: open. The live targets for this talk.
